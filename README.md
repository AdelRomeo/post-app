# Приложение написанное на React. Представляет из себя страничку со списком постов
## Реализован функционал:
* Подгрузка списка постов с сервера
* Удаление постов
* Добавление новых постов
* Возможность отмечать посты как понравившиеся
* Возможность отмечать посты как важные
* Фильтр постов (Показ всех постов либо только тех у которых есть отметка 'понравившиеся')
* Поиск по всем постам и отображение на странице только тех которые удовлетворяют поиску
